# Local AI Usage — Starting Point

I am using LM Studio on Windows 11 with an RTX 4080 to explore practical local AI use.

Current working setup:

* LM Studio can run local chat and embedding models.
* `jray/big-rag` is installed as a normal LM Studio plugin.
* Big RAG can search the Ancient Egypt and the Bible project and provide relevant passages to a local model.
* Project documents: `C:\Workspaces\ancient-egypt-and-the-bible`
* Vector index: `C:\Workspaces\LMStudioData\ancient-egypt-and-the-bible-rag`
* Embedding model: `nomic-ai/nomic-embed-text-v1.5-GGUF`
* Retrieval limit: `8`
* Affinity threshold: `0.48`
* Chunk size: `384`
* Chunk overlap: `80`

The next discussion should focus on choosing useful local models, balancing speed and quality, VRAM and context-size limits, coding versus general-analysis models, and practical workflows that combine a local model with Big RAG.
